# 프로젝트 메모리

## 목적

- `kasi-lunar`는 한국 음력/양력 변환을 위한 TypeScript npm 라이브러리
- `korean-lunar-calendar`의 확장 레이어로 구성 중

## 안정적인 범위

- 실제 작업은 `kasi-lunar/` 안에서만 수행
- 다른 열린 폴더들은 참조 전용
- 출력은 npm 라이브러리 형태 유지: 타입 익스포트, 빌드 가능한 dist, 스크립트 가능한 검증

## 현재 기준선

- 빌드 도구: `tsup`
- 타입 검사: `tsc --noEmit`
- 메인 패키지 진입점: `src/index.ts`
- 기존 검증 스크립트: `scripts/verify-build.mjs`

## 제약 사항

- 무거운 프로세스 문서 대신 작고 명시적인 하네스 파일 선호
- Codex 루프를 빠르게 유지
- 배포된 패키지 형태에 대한 호환성 기대치 보존
- 참조 폴더는 안전한 수정 대상이 아님
